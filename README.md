#Project: ES6 and HTML5 based Vide player written in Vanilla JS.

## How to run this?
- check out the latest release branch
- npm install
- npm run start

## How to test?
- npm run test

## Stack and major libraries
- Babel
- Webpack
- PostCSS
- Eslint
- Stylelint
